# Bo Pang Academic Homepage

This repository contains Bo Pang's personal academic website at <https://bopang2001.github.io>.

The site uses the Academic Pages Jekyll template. It is intended for academic exchange, postdoctoral or collaboration contacts, and a concise record of publications and research projects.

## Local Build

```bash
bundle config set --local path 'vendor/bundle'
bundle install
bundle exec jekyll build
```

## Content

- `_pages/about.md`: homepage and short bio
- `_publications/`: publication entries
- `_portfolio/`: research project entries shown as Projects
- `files/MyCV.pdf`: CV PDF
- `images/ieee.jpg`: profile image used in the sidebar
